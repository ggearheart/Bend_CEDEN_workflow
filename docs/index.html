Four-step pipeline

From raw Bend Genetics CSV to a validated CEDEN submission file.

1

Parse

Read all results CSVs from a single upload or a whole folder. Dates and times are normalised automatically.

01_parse_csv.R
2

Transform

Pivot wide → long, map Bend analyte names to CEDEN vocabulary, handle non-detects at the reporting limit.

02_transform_ceden.R
3

Validate

Check every required CEDEN field, ND conventions, and unmapped analytes. Issues listed before export.

03_validate.R
4

Export

Write the official CEDEN 2.0 Excel template or legacy workbook, plus flat CSVs. Downloadable as a zip.

04_export.R

Features

Everything needed to clear a multi-year backlog of FHAB submissions.

📂

Single file or batch folder

Upload one results CSV from an email attachment, or point to a folder and process every file at once.

⇄

Editable analyte mapping

View and edit Bend → CEDEN analyte names, units, and methods in-app. Changes save back to the lookup CSV.

📋

CEDEN 2.0 & Legacy

Output into the official CEDEN 2.0 Excel template (39-column Chemistry_Results sheet) or the legacy WaterChemistry format.

🔬

5% QA spot-check

Automatically samples 5% of output rows and runs 10 completeness and correctness checks. Results shown as a colour-coded report.

✅

Pre-export validation

Required fields, ND = RL consistency, unmapped analytes, and matrix codes all checked before a file is written.

⬇

One-click zip download

All output files (Excel + CSV) bundled into a single zip for submission or archiving.

Get started

Requires R 4.1+ and RStudio (recommended).

# 1. Clone the repo
git clone https://github.com/ggearheart/Bend_CEDEN_workflow.git
cd Bend_CEDEN_workflow

# 2. Drop Bend results CSVs into data/raw/
cp /path/to/20260610_SWAMP_CDFW_results.csv data/raw/

# 3. Launch the Shiny app (installs missing packages automatically)
Rscript launch_app.R

# — or run the headless pipeline —
Rscript run_workflow.R
